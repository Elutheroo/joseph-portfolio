OOU Portal Redesign

Role: UX/UI Designer | Duration: 1 Month

Tools: Figma, Sketchbook

Overview

The Olabisi Onabanjo University portal had poor navigation, an outdated layout, and accessibility issues. The goal was to create an intuitive, modern, and visually appealing experience for students, staff, and cyber cafe operators.

Process

  • Reviewed existing portal design and identified usability issues.
  • Created a quick mental user journey map to highlight problem areas.
  • Sketched wireframes in a notebook.
  • Moved wireframes into Figma for digital refinement.
  • Set UI design parameters and applied modern visual hierarchy principles.
  • Designed light and dark mode versions for accessibility.

Results

The redesigned portal offers clearer navigation, improved readability, and an overall better user experience for all user groups.
