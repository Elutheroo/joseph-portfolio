Olabisi Onabanjo University Portal Redesign

Transforming an outdated university portal into a modern, intuitive experience for students, staff, and cyber café operators.

Project Overview

Duration: 1 Month

Role: UX/UI Designer

The Problem: The portal was difficult to navigate, with an outdated and cluttered layout that frustrated users.

The Goal: Create a modern, intuitive interface improving usability for students, staff, and cyber café operators.

Process

  1. Evaluated Existing Portal – Reviewed the current design visually and functionally.
  2. Mapped User Flows – Created a quick mental journey map to identify navigation friction points.
  3. Paper Wireframes – Sketched layout improvements in a notebook.
  4. Digital Wireframes – Developed wireframes in Figma for structure and clarity.
  5. UI Design – Applied visual hierarchy, improved icons, and designed a dark mode option.

Accessibility & Usability Improvements

  • Simplified navigation with logical menu grouping
  • Clear visual hierarchy for faster scanning
  • Effective icon usage to improve recognition
  • Dark mode for contrast and visual comfort

Impact & Learnings

The redesign makes the portal cleaner, faster to navigate, and more inclusive for all user types.

Key takeaway: Even legacy systems can be revitalized with thoughtful UX improvements and modern UI practices.
