FinTrack Mobile App

Overview

FinTrack is a mobile app for personal finance management. I led the redesign to make the app more intuitive, accessible, and visually clean for everyday users.

Problem & Goals

Users were overwhelmed with financial data. The goal was to simplify navigation, provide quick insights, and enhance the visual hierarchy for better readability.

Process

- Conducted user interviews and persona mapping
- Sketched user flows and created wireframes
- Developed UI components using atomic design
- Prototyped transitions and interactions in Framer

Results

Time spent completing key actions dropped by 40%, and new users reported a 90% satisfaction rate with the updated app interface.

Tools Used

Sketch, Framer, Zeplin, Notion

Key Learnings

Designing for simplicity requires deep empathy—clear feedback, spacing, and contrast made the biggest difference in user experience.
