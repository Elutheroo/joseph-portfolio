OOU Portal Redesign

Making the university portal intuitive and accessible for all user groups

Overview

The OOU Portal was outdated, difficult to navigate, and lacked a clear structure for different user groups. This project focused on improving usability, navigation, and aesthetics while introducing a dark mode option.

Problem & Goals

Problem: Poor layout, outdated design, and confusing navigation made it frustrating for students, staff, and cyber cafe operators to use the portal.

Goals: Create an intuitive interface, enhance visual hierarchy, and provide a seamless experience for all user types.

Duration

1 Month

Role

UX/UI Designer

Process

  • Analyzed the existing portal design.
  • Mapped out a quick mental user journey to identify pain points.
  • Created paper wireframes to explore layout improvements.
  • Designed high-fidelity mockups in Figma with improved hierarchy, icons, and dark mode.

Results

The redesign improved clarity, reduced time spent navigating, and offered a modern aesthetic that matched user expectations.

Tools Used

Figma, Sketchbook, Miro
