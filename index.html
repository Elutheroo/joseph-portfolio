Featured Projects

OOU Portal — Student Portal Redesign

Redesigned the Olabisi Onabanjo University portal to simplify navigation, improve information architecture, and make essential tasks — checking results, printing exam passes, paying fees — far easier and more inclusive for students, staff, and cyber-café operators.

Theo’s Bakery — Mobile E-commerce

End-to-end mobile app design for a bakery e-commerce experience: onboarding, browsing, secure checkout, and order flow. Built as a course project using the full design-thinking cycle — research, iterate, prototype, test.

Study Room — Collaborative Learning

A flexible study and knowledge-management app that lets students create subject rooms, add resources (images, audio, documents, video), track study progress, set reminders, and invite peers to collaborate.
